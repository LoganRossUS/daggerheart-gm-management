Daggerheart GM Tools

Tools for running Daggerheart sessions

⚔
GM Control Panel
Main control center for GMs. Manage encounters, battle maps, tokens, fear tracker, dice roller, and more.
📺
Player Display
Full-screen display for players. Shows spotlighted adversaries, NPC portraits, and syncs with the GM panel.
🗺
Battle Map
Tactical battle map display with tokens, grid overlay, and real-time sync from the GM panel.
👾
Encounter Generator
Build and export encounters with adversaries. Search the bestiary and create encounter JSON files.

Daggerheart is a tabletop RPG by Darrington Press
